## Core commands 

- git init
- git commit
- git add ( Also to track new/modified files - "add precisely this content to the next commit")
- git clone <url> <new directory name>
- git status
- git rm
- git mv
- git log (--patch/ --stat/ --pretty)
- git log --pretty=format
- git checkout
- git remote add origin <> (push)
- git push -u origin main
